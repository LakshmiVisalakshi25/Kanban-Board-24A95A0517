# Kanban Board Project

## Overview
This is a fully interactive **Drag-and-Drop Kanban Board** built with **React**, **Tailwind CSS**, and **dnd-kit**.  
It allows users to manage tasks efficiently across different columns: **To Do**, **In Progress**, **Review**, and **Done**.  

The application supports **task creation, editing, deletion, drag-and-drop reordering**, and **persistence** using localStorage. It is also **keyboard-accessible** and includes ARIA live regions for screen readers.

---

## Features
- Add, edit, and delete tasks with a modal form.
- Assign **assignee** and **priority** to tasks.
- Drag tasks between columns or reorder within the same column.
- Search and filter tasks by title, assignee, or priority.
- Keyboard navigation and ARIA live region updates for accessibility.
- Task data is **persisted in localStorage** between sessions.
- Responsive layout for desktop and mobile devices.

---

## Technologies Used
- **React** for building the UI
- **Tailwind CSS** for styling
- **dnd-kit** for drag-and-drop functionality
- **localStorage** for data persistence
